# Logbook Pro SA

Logbook Pro SA is a South African vehicle logbook and fleet management app designed for SARS travel claims.

## Features

- SARS logbook tracking
- Fleet management
- Driver management
- Claims tracking
- Tyre tracking
- Maintenance tracking
- Document vault
- Backup & restore

## Built With

- React
- Vite
- Vercel

## Version

1.0.0 Beta
